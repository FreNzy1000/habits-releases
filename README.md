# Habits — Public Beta Releases

Публичная страница загрузки закрытой по смыслу, но технически публичной beta Habits.

Исходный код, ключи подписи, конфигурация Supabase и внутренняя документация здесь не публикуются.

## Скачать приложение

Последняя версия:

[Открыть beta-релизы](https://github.com/FreNzy1000/habits-releases/releases)

Для обычной установки нужен файл вида:

```text
Habits-0.3.0-beta.1.apk
```

Файл `.aab` не предназначен для прямой установки на телефон.

## Текущая версия

| Параметр | Значение |
|---|---|
| Версия | `0.3.0-beta.1` |

| Минимальный Android | Android 8.0 / API 26 |

## Установка

1. Откройте страницу последнего релиза.
2. В разделе **Assets** скачайте APK.
3. Разрешите установку приложений из этого источника, если Android попросит.
4. Откройте APK и завершите установку.
5. После установки можно снова отключить разрешение установки из неизвестных источников.

Скачивайте APK только с этой страницы GitHub Releases.

## Обновление

1. Откройте в Habits `Настройки → Данные`.
2. Создайте JSON-экспорт.
3. Выполните облачную синхронизацию.
4. Скачайте новый APK.
5. Установите его поверх текущей версии.

Не удаляйте старую версию перед обновлением, если это специально не указано в release notes.

## Проверка файла

Возле APK публикуется файл `.sha256`. Контрольную сумму можно проверить в PowerShell:

```powershell
Get-FileHash .\Habits-0.3.0-beta.1.apk -Algorithm SHA256
```
