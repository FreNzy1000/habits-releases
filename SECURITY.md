# Security Policy

## Сообщение о проблеме

Не публикуйте уязвимость вместе с рабочими ключами, экспортом базы или личными данными.

В сообщении укажите:

```text
Версия Habits
Модель устройства
Версия Android
Краткое описание
Шаги воспроизведения
Возможные последствия
```

## Не прикладывать

- пароли;
- токены;
- JSON-экспорт;
- чужие email;
- скриншоты с лишними личными данными;
- конфигурационные файлы;
- APK из неофициального источника.

Официальные APK публикуются только в GitHub Releases этого репозитория.
